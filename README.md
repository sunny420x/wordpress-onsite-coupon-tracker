## ระบบสร้างและติดตามการใช้งานคูปองหน้าร้าน ##
เคยอยากสร้างคูปองส่งเสริมการขายหน้าร้านแล้วไม่มีระบบไหม ? ปลั้กอินนี้ช่วยคุณได้

## คุณสมบัติ ##
สามารถสร้างแคมเปญ และคูปองในแคมเปญได้ คูปองที่ลูกค้าเก็บจะถูกแสดงในหน้า /my-account/ เมื่อลูกค้ามาแสดงคูปอง E-Voucher ที่หน้าร้าน พนักงานสามารถเปลี่ยนสถานะของคูปองนั้น ๆ ตามรหัสคูปองได้

## การใช้งาน ##
นำ ShortCode [evoucher_page] ไปวางในหน้าที่ต้องการให้แสดงจุดเก็บคูปอง และสร้างแคมเปญใหม่ได้เลย !
